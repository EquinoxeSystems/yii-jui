Class EDialog

Description
Description | Vars (details) | Methods (details)

EDialog is a Yii widget which encapsulates the functionality of the jQuery UI dialog widget to generate a dialog.

Works with jQuery 1.3 and jQuery UI 1.7+

http://jqueryui.com/demos/dialog/
