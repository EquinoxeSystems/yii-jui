Class ETabs

Description
Description | Vars (details) | Methods (details)

ETabs is a Yii widget which encapsulates the functionality of the jQuery UI tabs widget to generate a tabbed panel.

Works with jQuery 1.3 and jQuery UI 1.7

  • author: Rodolfo González González
  • link: http://docs.jquery.com/UI/Tabs
  • since: 1.0.2
