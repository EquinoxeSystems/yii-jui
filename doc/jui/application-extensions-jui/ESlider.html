Class ESlider

Description
Description | Vars (details) | Methods (details)

ESlider is a Yii widget which encapsulates the functionality of the jQuery UI slider widget to generate a dialog.

Works with jQuery 1.3 and jQuery UI 1.7

  • author: Rodolfo González González
  • link: http://docs.jquery.com/UI/Slider
  • since: 1.0.2
