Class EJqueryUiInclude

Description
Description | Methods (details)

EJqueryUiInclude is a class which can be used to include the jQuery UI } library without the need of including a widget in your view. This is useful in case you need to use just an effect, for instance.

This class is intended to work standalone. Don't use this class if you're going to use a jQuery UI widget wrapper.
